Semana 4: Programación Front-End con JavaScript y TypeScript

Descripción de los temas aprendidos

Durante la cuarta semana, se profundizó en el desarrollo front-end a través del uso de los lenguajes JavaScript y TypeScript, comprendiendo su importancia como pilares en la interactividad y lógica del lado del cliente en aplicaciones web modernas. Se estudió la estructura y sintaxis básica de JavaScript: declaración de variables, operadores, tipos de datos (números, cadenas, objetos, booleanos y otros) y estructuras de control como condicionales y bucles. En cuanto a funciones, se revisaron diferentes formas de declaración e invocación, incluyendo funciones anónimas, funciones como objetos, funciones flecha, auto-invocadas y closures, analizando sus usos según el contexto y necesidad. Además, se introdujo la manipulación del DOM (Document Object Model) para acceder, modificar y controlar elementos HTML dinámicamente mediante JavaScript. Asimismo, se exploraron las capacidades gráficas de Canvas, permitiendo desarrollar gráficos y animaciones interactivas desde el navegador. Estos aprendizajes fueron aplicados en el Laboratorio 04, donde se diseñó y programó una animación funcional utilizando JavaScript y el lienzo Canvas.

¿Qué aprendí?

Aprendí a programar scripts dinámicos en el navegador utilizando JavaScript y TypeScript, así como a manipular el contenido y estructura de las páginas web en tiempo real mediante el DOM. Comprendí los principios fundamentales de la programación funcional en el entorno web, además de aplicar gráficos animados utilizando la API de Canvas, lo cual expandió mis capacidades en la construcción de experiencias visuales más ricas e interactivas.

¿Cómo lo aprendí?

El aprendizaje fue progresivo y práctico. Se emplearon ejemplos orientados a la resolución de problemas reales y ejercicios guiados donde desarrollé funcionalidades concretas para páginas web. La implementación de una animación interactiva en el Laboratorio 04 me permitió aplicar de manera directa los conceptos sobre funciones, manipulación del DOM y uso de Canvas, consolidando mi comprensión mediante la práctica y la experimentación.
