Semana 7: Uso de Hooks en React y Desarrollo de Aplicaciones con Estado Compartido

Descripción de los temas aprendidos

Durante esta semana, se abordó uno de los pilares más importantes del desarrollo moderno con React: los Hooks. Estos permiten a los componentes funcionales mantener y gestionar estados, efectos secundarios, referencias, memorias y lógica compartida sin necesidad de clases. Se exploraron los siguientes hooks esenciales: useState: para declarar y actualizar estados locales en un componente. useEffect: para manejar efectos secundarios como peticiones a APIs o temporizadores. useContext: para compartir datos globales (como el tema, idioma o autenticación) sin necesidad de pasar props manualmente. useRef: para acceder a referencias de elementos del DOM o valores persistentes entre renderizados. useReducer: para gestionar estados complejos o estructuras tipo máquina de estados. useCallback y useMemo: para optimizar el rendimiento evitando recreaciones innecesarias de funciones o cálculos. Además, se introdujo la creación de Hooks Personalizados, permitiendo encapsular lógica reutilizable y mantener un código más limpio y modular. El Laboratorio 06 consistió en desarrollar una aplicación interactiva que utilizaba diversos hooks para gestionar el ciclo de vida, compartir estado entre componentes, acceder al DOM, y optimizar el rendimiento. También se aplicaron Hooks personalizados para encapsular comportamientos reutilizables. Finalmente, se realizó la Evaluación de Logro 01, en la cual se evaluó la comprensión y aplicación de los hooks, junto con una implementación práctica dentro de un entorno React.

¿Qué aprendí?

Aprendí a utilizar los React Hooks para mejorar el control del estado y el ciclo de vida de los componentes, así como a compartir lógica de negocio entre componentes mediante hooks personalizados. Estos conocimientos permiten crear aplicaciones más robustas, mantenibles y eficientes. También desarrollé habilidades en la gestión global del estado y referencias al DOM, así como en la optimización del rendimiento mediante memoización de funciones y valores.

¿Cómo lo aprendí?

El aprendizaje se realizó mediante el desarrollo del Laboratorio 06 y ejercicios guiados, donde implementé ejemplos prácticos utilizando cada hook. Se propusieron problemas que requerían el uso de múltiples hooks combinados, lo cual facilitó la comprensión de cómo estos interactúan entre sí. La evaluación de logro me permitió consolidar estos conceptos bajo presión controlada y demostrar el dominio de los mismos.
