Semana 6: Desarrollo de Aplicaciones con Eventos, Condicionales y Consumo de APIs en Framework JS

Descripción de los temas aprendidos

Durante esta semana se profundizó en las capacidades interactivas del desarrollo front-end con React.js, centrando el enfoque en el manejo de eventos, el renderizado condicional e iterativo, el trabajo con formularios, y el consumo de APIs externas para la obtención dinámica de datos. Se comprendió cómo gestionar eventos del usuario (como clics, entradas de texto, cambios de estado), así como su vinculación a funciones que modifican el estado interno del componente. Se trabajó el renderizado condicional, para mostrar elementos solo bajo ciertas condiciones lógicas, y el renderizado iterativo, para recorrer y mostrar listas dinámicamente, utilizando métodos como .map(). Además, se introdujeron los conceptos de promesas y funciones asíncronas con async/await, facilitando el consumo de servicios externos mediante la librería Axios, lo cual permitió obtener datos reales de una API y mostrarlos en la interfaz. También se presentó una introducción al enrutamiento (routing) en React, permitiendo la navegación entre diferentes vistas de la aplicación. El Laboratorio 06A consistió en implementar una interfaz que, al ejecutarse, capturaba eventos del usuario, consultaba una API externa en tiempo real, y mostraba la información recuperada utilizando renderizado condicional e iterativo.

¿Qué aprendí?

Aprendí a controlar la interacción del usuario mediante eventos y formularios en React, así como a representar dinámicamente el contenido de una aplicación web mediante condiciones y listas iterables. Además, aprendí a realizar peticiones HTTP a APIs externas utilizando Axios, gestionando las respuestas con async/await. Estos conocimientos me permiten construir aplicaciones que interactúan con datos en tiempo real y ofrecen una experiencia dinámica al usuario.

¿Cómo lo aprendí?

El proceso de aprendizaje fue eminentemente práctico. A través del Laboratorio 06A, se implementaron múltiples funcionalidades clave: captura de eventos, control de flujo lógico con renderizado condicional, recorrido de datos con métodos de iteración y consumo de APIs reales. Mediante ejercicios guiados y pruebas individuales, pude interiorizar la lógica de ejecución asincrónica, así como desarrollar aplicaciones que responden al usuario y a servicios externos de manera eficiente.
