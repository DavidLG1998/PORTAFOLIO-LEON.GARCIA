Semana 5: Desarrollo Front-End con Framework JavaScript (React.js)

Descripción de los temas aprendidos

En esta semana se abordó el desarrollo de aplicaciones front-end utilizando un framework JavaScript moderno, centrando el aprendizaje en React.js como herramienta para construir interfaces de usuario mediante componentes reutilizables. Se estudió el enfoque de Client Side Rendering (CSR), el cual permite renderizar contenido dinámico directamente en el navegador, mejorando la experiencia del usuario. Se exploró la estructura base de un proyecto en React, comprendiendo la gestión de dependencias a través de npm o yarn, así como el uso de herramientas como Vite o Create React App para la generación de proyectos. Se profundizó en la arquitectura basada en componentes, utilizando JSX como sintaxis extendida, y conceptos fundamentales como props, children, y el ciclo de vida de un componente. También se revisaron múltiples formas de aplicar estilos dentro del framework: desde estilos en línea (inline), hojas de estilo externas (CSS) y CSS Modules, hasta estilos gestionados desde componentes con librerías como Styled Components o la integración con Frameworks CSS como Tailwind. En el Laboratorio 05A, se aplicó el concepto de paso de datos entre componentes (props) y el uso de children para renderizar contenido dentro de un componente hijo, comprendiendo la lógica de flujo de datos en React.

¿Qué aprendí?

Aprendí a estructurar y desarrollar una aplicación web moderna utilizando un framework JavaScript (React.js), aplicando la filosofía de desarrollo basado en componentes. Comprendí cómo pasar datos entre componentes de forma jerárquica mediante props, y cómo encapsular contenido reutilizable con children. También adquirí habilidades para aplicar diferentes estrategias de estilización en componentes, dependiendo del contexto del proyecto.

¿Cómo lo aprendí?

El aprendizaje se realizó combinando teoría y práctica. A través de ejercicios guiados, implementé componentes funcionales que intercambian datos entre sí, experimentando con diferentes métodos de estilizado. El Laboratorio 05A fue fundamental para afianzar el uso de props y children, permitiéndome observar cómo se comunican los componentes en una aplicación React, y reforzando los principios de modularidad, reutilización y escalabilidad del código.
