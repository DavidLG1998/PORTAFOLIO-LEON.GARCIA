Semana 1: Introducción a las Tecnologías Web y Herramientas de Desarrollo

Descripción de los temas aprendidos

Durante esta primera semana, se abordaron los fundamentos esenciales de las tecnologías web, con el objetivo de comprender la estructura y funcionamiento del ecosistema digital actual. Se exploraron los distintos tipos de soluciones web, tales como sistemas web, aplicaciones web, sitios web y páginas web, así como los lenguajes, gráficos y protocolos que hacen posible su funcionamiento. Se introdujeron conceptos clave como DNS, TCP/IP y HTTP, los cuales permiten la comunicación eficiente en Internet. Finalmente, se identificaron los roles principales en el desarrollo de aplicaciones para la web: desarrollador frontend, backend y fullstack. Como complemento práctico, se utilizó la herramienta de desarrollo Visual Studio Code para familiarizarse con su interfaz, extensiones y atajos de teclado mediante el Laboratorio 01A.

¿Qué aprendí?

Aprendí a diferenciar las diversas soluciones web existentes, así como los componentes tecnológicos que hacen posible su creación y funcionamiento. Comprendí los principios básicos que permiten que una página web se visualice desde un navegador, y los actores clave involucrados en su desarrollo. También logré familiarizarme con el entorno de desarrollo Visual Studio Code, reconociendo su importancia en la creación eficiente de código para la web.

¿Cómo lo aprendí?

El proceso de aprendizaje se desarrolló de forma teórico-práctica. A través de las sesiones magistrales comprendí los conceptos fundamentales del desarrollo web y, mediante la realización del laboratorio, pude experimentar de forma práctica el uso de Visual Studio Code. Esta metodología me permitió reforzar los conocimientos adquiridos y visualizar cómo estos se aplican en un entorno profesional real.
