Semana 3: Desarrollo de Interfaces Front-End con Bootstrap y Tailwind CSS

Descripción de los temas aprendidos

En la tercera semana del curso, se abordó el desarrollo de interfaces web front-end utilizando dos potentes librerías de estilos: Bootstrap y Tailwind CSS. Se exploraron los layouts y componentes que ambas herramientas ofrecen para facilitar la construcción de sitios web visualmente atractivos, consistentes y responsivos. Con Tailwind CSS, se profundizó en el uso de su sistema de utilidades para el manejo de tipografías, líneas, textos, fondos, bordes, efectos visuales, tablas y espaciados. Además, se revisaron elementos fundamentales del diseño como container, columns, breakpoints, display, position, y se practicaron principios de Flexbox incluyendo dirección, envoltura, crecimiento y reducción de elementos. Se estudió también la creación de interfaces modernas con componentes como avatares, badges, dropdowns, botones y grupos de botones, así como estructuras de aplicación como shells, encabezados, visualización de datos, listas, formularios, retroalimentación, navegación, overlays, y layouts adaptativos. Estos conceptos fueron aplicados en la Práctica Calificada 03, donde se desarrolló un sitio web utilizando exclusivamente CSS, aplicando los principios aprendidos para lograr una interfaz limpia, accesible y funcional.

¿Qué aprendí?

Aprendí a utilizar frameworks modernos como Bootstrap y Tailwind CSS para el diseño y desarrollo eficiente de interfaces web. Comprendí cómo estructurar componentes reutilizables, cómo manejar el diseño responsivo mediante clases utilitarias y cómo aplicar estilos consistentes de forma rápida. Asimismo, reforcé mis conocimientos sobre Flexbox, posicionamiento y espaciado, lo cual me permite construir interfaces dinámicas y adaptables.

¿Cómo lo aprendí?

El proceso de aprendizaje fue práctico y progresivo. A través de ejercicios guiados y ejemplos reales, exploré la sintaxis y estructura de cada framework. Posteriormente, mediante la realización de la Práctica Calificada 03, consolidé lo aprendido diseñando un sitio web que integraba componentes visuales con estilos personalizados y adaptativos. El trabajo constante en Visual Studio Code y la retroalimentación recibida me permitieron identificar buenas prácticas y mejorar mi enfoque en el diseño web.
