Semana 2: Diseño Web con HTML, XML y CSS

Descripción de los temas aprendidos

Durante la segunda semana, se abordaron los fundamentos de construcción de páginas web utilizando HTML (Lenguaje de Marcado de Hipertexto) y XML (Lenguaje de Marcado Extensible) como parte del estándar de la Open Web Platform. Se exploró la estructura semántica de HTML5, el uso de sus principales etiquetas y la estructura jerárquica del Árbol DOM (Document Object Model), incluyendo su renderizado y el ciclo de vida de una página web. También se introdujeron conceptos clave como la optimización para motores de búsqueda (SEO) y la compatibilidad entre navegadores.Asimismo, se profundizó en el diseño visual de las páginas mediante CSS3, abarcando su sintaxis, tipos de visualización (block, inline, inline-block), unidades de medida absolutas y relativas, y los principales métodos de maquetación como Flexbox y Grid Layout. También se desarrollaron técnicas para lograr diseños fluidos y web responsiva, fundamentales para garantizar una experiencia adecuada en distintos dispositivos. Estas competencias se consolidaron con el desarrollo práctico del Laboratorio 02A, centrado en la creación de diseños flexibles y adaptativos.

¿Qué aprendí?

Aprendí a estructurar correctamente una página web utilizando HTML5 y XML, comprendiendo cómo se organiza y representa su contenido mediante el Árbol DOM. Identifiqué las prácticas recomendadas para mejorar el posicionamiento SEO y asegurar la visualización de la web en diversos navegadores. Además, aprendí a aplicar estilos profesionales usando CSS3, incluyendo técnicas modernas como Flexbox y Grid para diseñar interfaces responsivas que se adaptan a diferentes resoluciones de pantalla.

¿Cómo lo aprendí?

El aprendizaje se dio a través de una combinación de teoría y práctica. Las clases explicaron las bases del marcado HTML y XML, así como las mejores prácticas en diseño web. Posteriormente, en el Laboratorio 02A, pude aplicar los conocimientos adquiridos creando estructuras web funcionales y estilos modernos. El uso de ejemplos guiados y ejercicios prácticos me permitió afianzar los conceptos de forma progresiva y desarrollar habilidades clave en el diseño web actual.
